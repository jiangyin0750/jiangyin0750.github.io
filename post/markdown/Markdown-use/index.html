语法讲解

以st=>start为例:st为变量名,start为操作模块名，冒号后面为显示的标题。标题与冒号之间有空格
标题后面也可以加超链接如st=>start: 开始:>https://www.baidu.com/点图标跳转

所有操作模块名

操作模块 说明
start 开始
end 结束
opration 普通操作块
condition 判断块
subroutine 子任务块
inputoutput 输入输出块

st->op->cond中用变量名指向另一个变量名->≈箭头
cond(yes)->e cond(no)->op这两个中的yes,no只能用英文。类似if判断
