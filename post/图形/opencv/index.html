图像相关的API

imread - 读取图片

函数原型:Mat imread( const String& filename, int flags = IMREAD_COLOR )
参数:

  • const String& filename 读取图片的路径
    路径分为相对路径和绝对路径
    使用相对路径要保证图片在程序的源文件路径下，如果未包含在目录下的文件内直接写文件名,否则"文件名\\图像名"
    绝对路径就是图像在计算机中的位置
  • int flags 载入的方式
    IMREAD_COLOR为默认参数,意为使用BGR通道,IMREAD_UNCHANGED意为添加使用alpha通道
    更多的方式在定义在ImreadModes的枚举变量中,可通过再VS中按CTRL加左键点击imread再用同种方式点击flags查看

返回值为Mat类型,因为需要用Mat类型来接受
示例:Mat img = imread("image.jpg")

namedWindow - 新建windows窗口

函数原型:void namedWindow(const String& winname, int flags = WINDOW_AUTOSIZE)
参数:

  • const String& winname 显示的window窗口名
  • int flags 载入的方式
    可以使用默认参数,具体参数可在VS中WindowFlags中看,查找方式与上面的ImreadModes一样

示例:namedWindow("输出");

imshow - 输出图像

函数原型:void imshow(const String& winname, InputArray mat)
参数:

  • const String& winname 显示的window窗口名
    如果在输出前没有使用namedWindow新建窗口则自己新建新建,如果新建了可以输出到相应的窗口中
  • InputArray mat 需要输出的图片

示例:imshow("输出窗口",image);

imwrite - 保存图片

函数原型:bool imwrite( const String& filename, InputArray img, const std::vector<int>& params = std::vector<int>());
参数:

  • const String& filename 输出到的路径
  • InputArray img 输出的图片
  • const std::vector<int>& params = std::vector<int>() 特定格式保存的参数编码,一般无需管他,有默认值

示例:imwrite("D:\\123.jpg",image); Tips:输出图像的扩展名可以和输入时的扩展名不一样,

waitKey - 等待键盘输入

函数原型:int waitKey(int delay = 0);
参数:

  • int delay 等待时间
    如果参数不为0,则以毫秒(ms)为单位进行等待

在显示完图片后需要在程序最后加上waitKey(0);,防止图片一闪而过看不见
此函数还可以获取键盘输入内容,使用方法int key = waitKey();
返回值为int类型的,输入的键位与返回的大小与ASCII码对应

cvtcolor - 色彩空间转换

函数原型:void cvtColor( InputArray src, OutputArray dst, int code, int dstCn = 0 )
参数:

  • InputArray src 输入图像
  • OutputArray dst 输出图像
  • int code 要转换的色彩空间,具体参数参阅ColorConversionCodes
  • int dstCn 图片的通道数,0则为自动获取通道数

示例: cvtColor(img,dst,COLOR_BGR2GRAY);

使用Mat创建空白图像

Mat绘制空白图像的API有很多,具体可以看Mat类的构造函数
函数原型:zeros(int rows, int cols, int type)
参数:

  • int rows 宽度
  • int cols 高度
  • int type 类型
    这里的type需要是opencv中预定义的, 具体的格式在interface.h中
    构成有CV_<比特数>(S|F|U)C<通道数>
    比特数看图片的像素大小进行选择
    S是有符号整型,F是单精度浮点型,U是无符号整型
    <通道数> 1为灰度图片,3为三通道图片,4为四通道图片

示例:Mat m = Mat::zeros(Size(600,800),CV_8UC3) rows和cols可以使用Size包含
zeros可以改为ones这样得到的图片每个像素点的第一个通道值为1,zeros则为全0
使用cout<<m<<endl;可以得到图片的具体像素值
Mat常用方法
CopyTo(OutputArray dst, InputArray mask): 第一个参数是输出的对象,函数有重载mask可加可不加,加了则将mask中的非零像素点进行复制
Mat::channels(): 会返回图像的通道数
Mat::rows: 非函数,值是图像的行数,即横着的像素值
Mat::cols: 非函数,值是图像的列数,即竖着的像素值
empty(): 用来判断图片是否读取成功,成功返回0,失败返回1
Mat::at: 用来输出指定位置的像素点,用法Vec3b bgr = m1.at<Vec3b>(row, col);此为三通道图片的获取,获取后bgr[0]则为此像素的第一个通道,bgr[1]则是第二个

flip - 图片翻转

函数原型:void flip(InputArray src, OutputArray dst, int flipCode)
参数:

  • int flipCode
    ROTATE_180 = 1: 镜像左右翻转
    -1: 顺时针旋转180度
    0: 逆时针旋转180度

示例:flip(img[0], img[1], 1);

addWeighted - 图像线性混合

效果是两个图层进行融合
函数原型:(InputArray src1, double alpha, InputArray src2,double beta, double gamma, OutputArray dst, int dtype = -1)
参数:

  • double alpha 图片的alpha值(透明度/权重),两张图片分别对应不同的透明度,另一个即beta
  • double gamma 偏差值,正加负减

混合之后的图像BGR会根据两图片的alpha值和各自的每个像素BGR值进行计算
B1,B2分别指两图各像素的B通道值
((B1*alpha1+B2*beta+gamma),(G1*alpha1+G2*beta+gamma),(R1*alpha1+R2*beta+gamma))
如一个图片的BGR为(20,125,255)和一个BGR为(40,50,70),且alpha值都为0.5的情况下,混合后的BGR为(30,88,162),即((200.5+400.5),(1250.5+500.5),(2550.5+700.5))
如果gamma值是50,其BGR会变成(80, 138, 212),为负值则减超过阈值则保留至0或255,如-50,则(0,38,112)
Tips: 计算后的值会进行四舍五入
示例:addWeighted(img[0],0.5,img[1],0.5,0,dst)

四则运算图像混合

分别是:

  • add() 加法运算
  • subtract() 减法运算
  • multiply() 乘法运算
  • divide() 除法运算

这四种混合方法的效果并不是很好,具体参数可参阅函数的声明

blur - 线性滤波

函数原型: void blur( InputArray src, OutputArray dst,Size ksize, Point anchor = Point(-1,-1),int borderType = BORDER_DEFAULT )
参数:

  • Size ksize 内核大小,取值取奇数,便于找描点
  • Point anchor = Point(-1,-1) 描点的位置,(-1,-1)就是中心,一般无需传参
